# 🎙️ N.I.K.K.I TTS Studio - Frontend

Modern React-based frontend for the N.I.K.K.I Text-to-Speech Studio.

Convert text into natural-sounding speech using AI-powered neural voices.

---

## ✨ Features

* 🎨 Modern responsive UI
* 🎤 Generate speech instantly
* 🌍 Multiple voice selections
* 🎚️ Rate, Pitch & Volume controls
* ▶️ Built-in audio player
* ⬇️ MP3 download support
* 🌙 Clean dark theme
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* React
* Vite
* JavaScript
* CSS3
* Fetch API

## 🚀 Installation

Clone repository:

```bash
git clone <repo-url>
cd frontend
```

Install dependencies:

```bash
npm install
```

---

## ⚙️ Environment Variables

Create:

```text
.env
```

Add:

```env
VITE_API_URL=http://localhost:8000
```

For production:

```env
VITE_API_URL=https://api.yourdomain.com
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

Default URL:

```text
http://localhost:5173
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
